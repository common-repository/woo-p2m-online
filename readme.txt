Плагин для оплаты через сервис pay2me.world

== Description ==
Интегрированный в Woocommerce плагин платежной системы pay2me.world

== Installation ==
1. Загрузить содержимое архива `woo-p2m-online.1.1.zip` в  директорию `/wp-content/plugins/`
2. Активируйте плагин через меню «Плагины» в WordPress
3. Перейдите в настройки Woocommerce -> Платежи -> P2MOnline, заполните все данные из личного кабинета (lk.pay2me.world)
